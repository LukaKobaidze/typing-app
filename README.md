- [Sass](https://sass-lang.com/)

## Getting Started

1. Clone the repo
```sh
git clone https://github.com/LukaKobaidze/typing-app
```
2. Install NPM packages
```sh
npm install
```
3. Run locally on localhost:3000
```sh
npm start
```
